# Analysis Checklist

## Before Analysis

- Define the business question.
- Check missing values and duplicates.
- Standardize dates, categories, and numeric fields.

## During Analysis

- Build pivot tables around the main KPIs.
- Compare trends over time and across categories.
- Separate descriptive findings from recommendations.

## After Analysis

- Create a one-page dashboard.
- Write three practical business conclusions.
- Note data limitations.
